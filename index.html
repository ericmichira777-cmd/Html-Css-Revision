Responsive web design

Design mobile-first, then let @media queries add breathing room on bigger screens. Resize the browser — this grid rewires itself at 640px and 960px.

Responsive checklist
  • Set the <meta viewport> tag.
  • Use relative units: rem, %, vw, clamp().
  • Prefer flex/grid over fixed widths.
  • Images: max-width: 100%; height: auto;.
  • Test with real thumbs, not just Chrome DevTools.
